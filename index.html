Now I'm the text that has paragraph tags around it, but why do we need paragraph tags in the first place?

Well paragraph tags create line breaks, just like me!

Without the paragraph tag, text would constantly look like this:

I don't have a paragraph tag. This is how text is presented without one. Constantly on the same line, with no formatting. So having paragraph tags makes your code and site look cleaner.

They do this by introducing a line break within the website.

Here is some more text
with a line break

Line breaks also moves your text to the next line of the webpage making your page look more professional.

This is bolded text. this is done using the "strong" tag. strong = bolded text.

this is italicized text. this is done using the "em" tag. em = italic text.

This is underlined some text. this is done using the "ins" tag. ins = underlined text.

This is super scripted text. this is done using the "sup" tag. sup = super scripted / above the line.

this is sub scripted text. this is done using the "sub" tag. sub = sub scripted / bellow the line text.

this is struck out text. this is done using "del" tag. del = deleted / struck out text.


The "hr" tag gives us the horizontal line above and bellow this text. This can be used as an alternatve line/page break and I've been using this tag to break up the different sections of this page.
